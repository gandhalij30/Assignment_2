()- event Binding syntax is used to pass the data to HTML to ts.
[]- Property binding syntax is used to pass the data from ts to HTML
[()]- this is called as banana inbox syntax which is used for 2 way data binding.
        but in order to use 2 way data binding we have to use a builtin module known as FormsModule 

Two way data binding is a very important feature for angular and very popular .
All form submissions are typically done using Two way data binding syntax.

[(ngModel)]="username" 
    where ngModel is a builtin attribute given by angular.
    username is a field defined in the ts file.
